# 📝 Todo List

A simple and responsive Todo List application built with HTML, JavaScript, and Tailwind CSS.

## Features

- Add, edit, and delete tasks
- Mark tasks as completed
- Dark & Light mode
- Responsive design
- Local Storage support

## Tech Stack

- HTML5
- JavaScript (ES6)
- Tailwind CSS

## Run Locally

```bash
npm install
npm run dev
```
